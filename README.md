# MY2DO 

[MY2DO](https://my-2-do.vercel.app/)

## Functions:
- Adding, editing and deleting tasks
- Adding and removing custom folders
- Sorting tasks alphabetically and by date added/changed
- Possibility of planning
- Dark and light theme
- Search

## Stack:
- JavaScript
- TypeScript
- React
- Redux-Toolkit
- Material UI library
- Vite
